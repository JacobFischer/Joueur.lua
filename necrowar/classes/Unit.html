Class Unit

A unit in the game.

May be a worker, zombie, ghoul, hound, abomination, wraith or horseman.

Fields

self.acted Whether or not this Unit has performed its action this turn (attack or build).
self.health The remaining health of a unit.
self.job The type of unit this is.
self.moves The number of moves this unit has left this turn.
self.owner The Player that owns and can control this Unit.
self.tile The Tile this Unit is on.
self.gameObjectName (inherited) String representing the top level Class that this game object is an instance of.
self.id (inherited) A unique id for each instance of a GameObject or a sub class.
self.logs (inherited) Any strings logged will be stored here.

Methods

Unit:attack (tile) Attacks an enemy tower on an adjacent tile.
Unit:build (title) Unit, if it is a worker, builds a tower on the tile it is on, only workers can do this.
Unit:fish (tile) Stops adjacent to a river tile and begins fishing for mana.
Unit:mine (tile) Enters a mine and is put to work gathering resources.
Unit:move (tile) Moves this Unit from its current Tile to an adjacent Tile.
Unit:log (message) (inherited) Adds a message to this GameObject's logs.

Methods

Unit:attack (tile)
Attacks an enemy tower on an adjacent tile.

Parameters:

  • tile Tile The Tile to attack.

Returns:

    bool True if successfully attacked, false otherwise.
Unit:build (title)
Unit, if it is a worker, builds a tower on the tile it is on, only workers can do this.

Parameters:

  • title string The tower type to build, as a string.

Returns:

    bool True if successfully built, false otherwise.
Unit:fish (tile)
Stops adjacent to a river tile and begins fishing for mana.

Parameters:

  • tile Tile The tile the unit will stand on as it fishes.

Returns:

    bool True if successfully began fishing for mana, false otherwise.
Unit:mine (tile)
Enters a mine and is put to work gathering resources.

Parameters:

  • tile Tile The tile the mine is located on.

Returns:

    bool True if successfully entered mine and began mining, false otherwise.
Unit:move (tile)
Moves this Unit from its current Tile to an adjacent Tile.

Parameters:

  • tile Tile The Tile this Unit should move to.

Returns:

    bool True if it moved, false otherwise.
